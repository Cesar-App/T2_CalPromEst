Nombre del Proyecto: Cálculo de la Nota Promedio de un Estudiante

Descripción: Este proyecto consiste en una aplicación de escritorio que permite registrar y así mismo le devuelve al usuario los datos correspondientes sobre el rendimiento academico del estudiante.

Tecnologías Utilizadas:

Lenguaje(s): C#
Herramientas / Framework: No aplica
Base de datos (si aplica): No aplica
Entorno o IDE: Visual Studio 2022
